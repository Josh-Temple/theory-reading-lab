# Style Guide (Lecture-Series Writing)

Tone: calm, serious, and teachable.

## Principles
1. **Clarity over compression**
   - Choose sentences that can be understood in one hearing.
   - Prefer plain conceptual scaffolding before nuance.

2. **One episode, one center**
   - Each episode should revolve around one central question or claim.
   - Supporting points should clarify the center, not compete with it.

3. **Separate source-based claims from interpretation**
   - Mark what is grounded in source text vs what is editorial interpretation.
   - Avoid presenting interpretation as direct citation.

4. **Avoid jargon stacking**
   - Introduce technical terms gradually.
   - Do not chain multiple abstract terms without examples.

5. **Audio-friendly prose when relevant**
   - Prefer shorter units and explicit transitions for spoken delivery.
   - Reduce parenthetical complexity and nested clauses in script drafts.
